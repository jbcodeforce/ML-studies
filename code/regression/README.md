# Regression

Linear, polynomial, and regularized regression with scikit-learn and Spark.

## Scripts

| Path | Description |
| --- | --- |
| `demo_lasso_ridge.py` | L1/L2 regularization comparison |
| `house-pricing/` | House price prediction |
| `kaggle-training/` | Kaggle competition examples |
| `spark_linear_regression.py` | Spark ML linear regression |

## Notebooks

`LinearRegression.ipynb`, `PolynomialRegression.ipynb`, `MultivariateRegression.ipynb`
