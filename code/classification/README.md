# Classification

Classical and neural classifiers: Adaline, SVM, decision trees, random forest, naive Bayes, and PyTorch NN examples.

## Scripts

| File | Description |
| --- | --- |
| `adaline.py` | Adaptive linear neuron |
| `svm_iris.py` | SVM on iris |
| `decision_tree_iris.py` | Decision tree |
| `random_forest_iris.py` | Random forest |
| `naive_bayes_classifier.py` | Naive Bayes |
| `nn-classifier.py` | PyTorch binary classifier |

## Notebooks

`classifications.ipynb`, `multiclass-classifier.ipynb`, `sklearn-classifiers.ipynb`, `KNN.ipynb`, `NaiveBayes.ipynb`, `DecisionTree.ipynb`, `SVC.ipynb`, `TrainTest.ipynb`, `KFoldCrossValidation.ipynb`, `spark_decision_tree.py`

## Environment

Notebooks: `jupyter/.venv` kernel (run `uv sync --extra pytorch` for PyTorch notebooks). Scripts: `code/.venv` (add `--extra pytorch` for NN scripts).
