# Complete AI/ML Learning Guide

## Summary

A structured, end-to-end roadmap for learning AI, ML, Deep Learning, Generative AI, and Agentic Systems. The guide integrates internal repository resources with external learning materials across five progressive phases.

## Phases

| Phase | Topic | Duration |
|-------|-------|----------|
| 0 | Prerequisites (Python, Git, Linux) | — |
| 1 | Mathematical Foundations (Linear Algebra, Probability, Calculus) | — |
| 2 | Core Machine Learning (Supervised, Unsupervised, Model Selection) | 2-3 months (beginner) |
| 3 | Deep Learning (Neural Networks, PyTorch, CNNs, Transfer Learning) | 3-4 months (intermediate) |
| 4 | LLMs, NLP, Generative AI (Transformers, RAG, Prompt Engineering) | — |
| 5 | Agentic Systems (Agents, LangGraph, Multi-Agent, MCP) | 3-4 months (advanced) |

## Key Connections

- **Phase 0** links to [Python ML Development Environment](../concepts/python-ml-development-environment.md) and coding resources.
- **Phase 1** references [Mathematical Foundations](../concepts/mathematical-foundations.md) and [Bias-Variance](../concepts/bias-variance.md).
- **Phase 2** covers core ML topics mapped to [ML Pipeline](../concepts/ml-pipeline.md) and [Scikit-learn](../concepts/scikit-learn.md).
- **Phase 3** builds on [PyTorch Library](../concepts/pytorch-library.md) and [Transformer Architecture](../concepts/transformer-architecture.md).
- **Phase 4** connects to [Generative AI Overview](../concepts/generative-ai-overview.md), [RAG Architecture](../concepts/rag-architecture.md), and LLM frameworks.
- **Phase 5** maps to [Agentic AI](../concepts/agentic-ai.md), [LangGraph](../concepts/langgraph.md), and [Model Context Protocol](../concepts/model-context-protocol.md).

## Learning Paths

Three recommended tracks: **Beginner** (2-3 months through Phase 2), **Intermediate** (3-4 months through Phase 4), and **Advanced** (3-4 months through Phase 5).
