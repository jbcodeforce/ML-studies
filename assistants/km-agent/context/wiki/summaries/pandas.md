# Pandas Summary

Pandas is a foundational Python library for data exploration and manipulation. It uses the `DataFrame` structure to represent tabular data with rows and columns. Pandas supports loading from many formats including CSV, Excel, SQL, JSON, and Parquet. Common operations include inspecting data with `describe()` and `head()`, handling missing values via `isnull()` and `dropna()`, and extracting feature/target arrays for ML libraries like scikit-learn. The library is essential for preprocessing data before feeding it into machine learning pipelines.

**Key Concepts**: DataFrame, data exploration, missing data handling, feature extraction, ML pipeline integration.
