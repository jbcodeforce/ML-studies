# Visualization of the Data

A brief source covering data visualization in ML workflows, focusing on **Matplotlib** as the primary Python plotting library.

Matplotlib is used across the ML studies codebase for key visualization tasks: plotting activation functions like the sigmoid, rendering dataset images (e.g., MNIST Fashion), and drawing decision boundaries for classifiers. Associated code includes the `sigmoid-play.ipynb` notebook for interactive exploration, `plotting-dataset.py` for dataset visualization, and `pytorch_helper_functions.py` for decision boundary plotting.

Matplotlib is part of the standard Python ML development environment alongside NumPy, SciPy, and Seaborn.
