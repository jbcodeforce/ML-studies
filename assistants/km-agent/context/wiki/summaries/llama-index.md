# LlamaIndex Library

LlamaIndex is a framework for building context-augmented LLM applications, including Q&A systems, chatbots, document understanding and extraction, and agentic apps. It provides data connectors, indexes, natural language engines, agents, and tools for observability and evaluation.

Notable sub-projects include **LlamaParse** (a document parsing solution) and **LlamaHub** (a repository of pre-built connectors). The framework is commonly compared to LangChain and Haystack, with a particular strength in retrieval-augmented generation (RAG) pipelines through its indexing-focused architecture.
