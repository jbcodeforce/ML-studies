# Data Scientist Skill Set

This document outlines the core competencies required for data science, organized into five categories: Mathematics, Statistics, Python, Data Visualization, and Machine Learning (including deep learning). Drawing from Udemy, LinkedIn, and other sources, it presents knowledge as a Q&A covering essential ML concepts.

Key topics include supervised vs. unsupervised learning, the bias-variance tradeoff, regularization techniques (L1, L2, Elastic Net), overfitting/underfitting, the end-to-end ML pipeline (problem definition through deployment and monitoring), classification vs. regression, common evaluation metrics (accuracy, precision, recall, F1, MSE, RMSE, R², AUC-ROC, MAP), cross-validation, and feature selection methods (filter, wrapper, embedded, ensemble). The document emphasizes that feature selection reduces overfitting, improves interpretability, cuts computational cost, and boosts accuracy—making it a vital step in any ML workflow.
