# Summary: Anthropic Notes

Anthropic is the company behind the Claude family of large language models. The current generation of Claude models includes **Haiku**, **Sonnet**, and **Opus** tiers, each targeting different use cases balanced across safety, accuracy, and security.

## Key Resources
- **Anthropic Console** ([console.anthropic.com](https://console.anthropic.com/dashboard)): Prompting interface for experimentation.
- **Claude.ai**: Chat interface for interacting with Claude.
- **Anthropic Cookbook** (GitHub): Reference examples and best practices.
- **Anthropic Prompt Library**: Curated prompt templates.
- **LangChain Integration**: LangChain provides integration support for Anthropic models.

## Model Tiers
| Model | Use Case |
|-------|----------|
| Claude 3 Haiku | Fast, cost-effective for simple tasks |
| Claude 3 Sonnet | Balanced performance for general use |
| Claude 3 Opus | Maximum capability for complex reasoning |
