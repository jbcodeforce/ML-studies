# Reinforcement Learning

Reinforcement learning aims to develop a system (agent) that improves its performance based on interactions with an environment.

## Key Concepts

- **Agent** — The system that learns and takes actions
- **Environment** — The world the agent interacts with
- **Reward** — Feedback signal the agent seeks to maximize
- **Exploration vs. Exploitation** — Balancing trying new actions versus using known good actions

## Learning Approach

The agent learns a series of actions that maximizes reward through:
- **Exploratory trial-and-error** — Trying actions and learning from outcomes
- **Deliberative planning** — Planning ahead based on learned models

Unlike supervised learning (which learns from labeled data) or unsupervised learning (which discovers structure), reinforcement learning learns through interaction and feedback.
