# Pydantic AI

Pydantic AI is an agent framework by the Pydantic team, bringing the developer experience of FastAPI to generative AI application development.

## Overview

Pydantic AI is designed to simplify building production-grade GenAI applications with a focus on type safety, structured data handling, and developer ergonomics.

## Value Propositions

- **Model Agnostic**: Supports a broad range of LLM providers including OpenAI, Anthropic, Gemini, DeepSeek, Ollama, Groq, Cohere, and Mistral. This allows teams to swap backends without rewriting application logic.
- **Type-Safe**: Built on Pydantic's validation ecosystem, ensuring reliable data contracts between application code and LLM interactions.
- **Structured Responses**: Enables typed, validated output schemas for LLM-generated content.
- **Streamed Responses**: Supports streaming for interactive, low-latency applications.

## Design Philosophy

Pydantic AI applies the same principles that made FastAPI successful—clear APIs, type hints, and developer-friendly tooling—to the domain of LLM-powered applications and agents.
