# Cohere Platform

Cohere is an enterprise AI platform offering large language models and NLP products designed for production deployments. The platform focuses on multilingual capabilities, RAG support, and easy customization through fine-tuning.

## Core Products

### Command-R
The flagship LLM, optimized for long-context enterprise tasks:
- **128K context length** for processing lengthy documents
- **Multilingual** support across 10 key languages
- **RAG with citation** support to reduce hallucinations
- **Tool use** capabilities enabling single-step and multi-step agent workflows

### Embed
Embedding models producing 4096-dimensional vectors for:
- Semantic similarity estimation between texts
- Text categorization and clustering
- RAG pipeline accuracy improvements
- Support for 100+ languages

### Rerank
Models that sort text by semantic relevance to a query:
- Injects language model intelligence into existing search systems
- Improves precision of search results
- Complements embedding-based retrieval in RAG pipelines

## Value Proposition

- **Enterprise-first** design with SaaS offering
- **No-code Playground** for rapid experimentation
- **Scalable models** with SDK and API integration
- **Strong NLP** with multilingual coverage
- **Open science** commitment via Aya, a 101-language multilingual model
