# Visualization of the Data

This source covers data visualization techniques for ML workflows, with a focus on **Matplotlib** — the foundational Python plotting library.

## Matplotlib

Matplotlib is the core plotting library in the Python ML ecosystem, referenced throughout the studies codebase for:

- **Getting started**: Quick-start guides and basic plotting patterns.
- **Interactive exploration**: Plotting sigmoid functions to understand activation behavior ([sigmoid-play.ipynb](https://github.com/jbcodeforce/ML-studies/blob/master/code/deep-learning/fundamentals/sigmoid-play.ipynb)).
- **Dataset visualization**: Rendering MNIST Fashion dataset images for inspection ([plotting-dataset.py](https://github.com/jbcodeforce/ML-studies/blob/master/code/deep-learning/fundamentals/plotting-dataset.py)).
- **Decision boundaries**: Helper utilities for visualizing classifier decision surfaces in PyTorch workflows ([pytorch_helper_functions.py](https://github.com/jbcodeforce/ML-studies/blob/master/code/shared/pytorch_helper_functions.py)).

Matplotlib integrates with other visualization libraries like Seaborn and is part of the standard Python ML development toolchain.
