# DIKW Pyramid

The DIKW pyramid is a visualization of the data value chain, showing how raw data transforms into wisdom through progressive enrichment.

## Levels

1. **Data** — The base of the pyramid. Inert facts and figures with the least perceived usefulness. Data is not self-organized or self-understandable.

2. **Information** — Data that has been processed by a program, machine, or person to add understanding. Information has higher value than raw data.

3. **Knowledge** — Information that has been organized and classified. Knowledge represents patterns, relationships, and context that give information practical value.

4. **Wisdom** — The highest perceived value. Wisdom combines data with progressive traits: perspective, context, understanding, learning, and the ability to reason.

## Moving Up the Chain

To advance from one level to the next requires:

- **Data → Information**: A program, machine, or person to add understanding.
- **Information → Knowledge**: Organization and classification.
- **Knowledge → Wisdom**: Integration of perspective, context, understanding, learning, and reasoning.
