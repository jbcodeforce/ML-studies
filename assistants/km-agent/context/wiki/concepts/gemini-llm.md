# Gemini LLM

Gemini is Google's multimodal large language model, accessible through Vertex AI. It accepts a wide variety of input modalities and produces text output.

## Capabilities

- **Multimodal Input**: Accepts text, images, video, audio, and document data as input
- **Text Output**: Produces text responses regardless of input modality
- **Vertex AI Integration**: Prompt and test Gemini within the Vertex AI platform
