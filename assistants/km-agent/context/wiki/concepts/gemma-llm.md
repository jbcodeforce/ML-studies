# Gemma LLM

Gemma is a family of lightweight, state-of-the-art open-weight large language models from Google, based on Gemini embeddings and using 256K token context.

## Model Characteristics

- **Parameter Sizes**: 9 billion and 27 billion parameter variants
- **Architecture**: Decoder-only text-to-text LLMs
- **Open Weights**: Both pre-trained and instruction-tuned variants available with open weights
- **Token Context**: 256K token context window
- **Small Device Target**: Designed to run on smaller devices

## Availability

- [Hugging Face](https://huggingface.co/google/gemma-2-9b)
- [Kaggle](https://www.kaggle.com/models/google/gemma)
